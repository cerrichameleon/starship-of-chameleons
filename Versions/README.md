# Runnable Versions

This folder stores runnable milestone artifacts for the Starship of Chameleons.

Purpose:
- preserve launchable snapshots for human testing
- make progress visible without digging through git history
- keep a small number of meaningful runnable versions

Rules:
- each version gets its own folder
- keep launch instructions inside each version folder
- artifacts should prefer runnable launch scripts or packaged outputs
- source history still lives in git; this folder is for human-facing runnable milestones

Naming suggestion:
- `v0.1.0-captain-chat-ui`
- `v0.2.0-dynamic-crew-proposal`
- `v0.3.0-runtime-console`
