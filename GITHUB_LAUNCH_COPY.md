## README Opening
# Starship of Chameleons Protocol

Starship of Chameleons Protocol is a structured multi-agent orchestration layer built on top of OpenClaw.

Its goal is to turn a loose collection of agents into a coherent team with:
- a Captain as the primary user-facing coordinator
- Specialists for focused execution
- a Chief Engineer for cost-aware model routing
- strategic Oxpecker quality control
- mission structure, memory, gates, and review flow

This project is being developed as a companion framework rather than a fork-first rewrite, so it can leverage OpenClaw's existing strengths while adding a stronger organizational and operational model on top.
