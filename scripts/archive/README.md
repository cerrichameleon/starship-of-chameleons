# Archived helper scripts

These are preserved helper scripts that were useful during Starship bring-up or debugging, but are not part of the main launcher surface.

## `run_starship_no_browser.py`

A preserved launcher variant that starts the onboarding-first web UI without trying to auto-open a browser.

Why keep it:
- it documents the no-browser-open workaround explicitly
- it was useful while browser auto-open behavior was being debugged on the host
- it may still be handy on headless or awkward host setups

This is archival/supporting material, not the primary recommended launcher.
